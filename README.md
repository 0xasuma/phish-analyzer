# Phishing Email Analyzer 🎣

AI-powered phishing detection tool. Paste email headers and body to get instant risk scoring, red flag detection, and social engineering tactic analysis.

## Features
- Risk score 0-100 with SAFE/SUSPICIOUS/PHISHING verdict
- Red flag extraction with explanations
- Suspicious URL detection and analysis
- Social engineering tactic identification
- Sender authentication analysis
- Dark-themed professional UI

## Setup
```bash
pip install -r requirements.txt
python app.py
```

Runs on http://localhost:5011
